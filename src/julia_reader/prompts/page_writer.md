### Deduplication Rules

When prior pages exist:

1. **Continue, don't repeat.** Begin where the last page ended. Do NOT repeat
   any material, topics, examples, or phrasing already covered in prior pages.
2. **Respect subject-shift boundaries.** If a boundary falls within your
   assigned section, use it as a natural page break. End the page there.
3. **New angles only.** Produce new analysis, new topics, or continued
   exposition. If all meaningful content has been covered, write a brief
   closing page rather than padding.
4. **Cross-reference.** Link back to specific prior pages where relevant
   (e.g., "As discussed on page_001.md…").
